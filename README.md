# REBWB - Real Estate Without Bullshit

A disciplined system for analyzing real estate deals without hype, gurus, or emotional decision-making.

## Structure

```
rebwb/
├── index.html          # Homepage
├── course.html         # Course curriculum (15 modules)
├── book-workbook.html  # Book & Workbook sales page
├── templates.html      # Professional templates
├── tools.html          # Interactive tools & calculators
├── faq.html            # FAQ
├── about.html          # About page
├── login.html          # Login
├── signup.html         # Signup
├── module-x-how-to-stop.html   # Module X: How to Stop
├── module15-tools.html         # Module 15 tools
├── css/                # Stylesheets
├── js/                 # JavaScript (auth, calculators, diagnostics)
├── audio/              # Audio narrations
├── video/              # Video content
├── workbooks/          # Downloadable PDF workbooks
├── REBWB_Book_Final.pdf
└── REBWB_Workbook_Final.pdf
```

## Tech Stack

- HTML/Tailwind CSS
- Supabase Auth
- Static deployment

## Deployment

Upload contents to any static hosting provider (Netlify, Vercel, GitHub Pages, etc.)
